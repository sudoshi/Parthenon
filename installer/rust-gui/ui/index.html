Community Stack

OMOP Data

Start with the database you already have. If you do not have one, the installer creates local PostgreSQL and uses Eunomia until a vocabulary is ready.

Local PostgreSQL will be installed with the Community stack. The OMOP CDM schema is omop, the vocabulary schema is vocab, and the results schema is results.
