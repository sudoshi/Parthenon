Before You Start

Docker is running

Use Docker Desktop on macOS or Windows, or Docker Engine with the Compose plugin on Linux.

Local resources are ready

Plan for at least 8 GB RAM and 20 GB of available disk for the Community stack.

Windows uses WSL 2

Open an Ubuntu or Debian WSL shell, enable Docker Desktop WSL integration, then run the same command.

What Happens Next

  1. Preflight checks Acropolis verifies Docker, Docker Compose, disk space, ports, and operating-system support.
  2. Community configuration The installer writes local environment files, chooses safe defaults, and keeps Enterprise-only prompts out of the MVP flow.
  3. Stack launch Docker Compose starts Parthenon and the Community sidecars, then the installer reports the local URL and admin setup path.

Direct Release Assets

Use these when a locked-down host cannot pipe the bootstrap script. Download the package for the host, verify it with the checksums file, then run the included installer binary.
