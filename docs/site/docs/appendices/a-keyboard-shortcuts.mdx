# Appendix A — Keyboard Shortcuts

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `?` | Open keyboard shortcut help overlay |
| `Ctrl/⌘ + K` | Open global command palette |
| `Ctrl/⌘ + /` | Toggle sidebar collapse |
| `Esc` | Close modal / dismiss panel |

## Navigation

| Shortcut | Action |
|----------|--------|
| `G D` | Go to Data Sources |
| `G V` | Go to Vocabulary Browser |
| `G C` | Go to Cohorts |
| `G A` | Go to Analyses |
| `G E` | Go to Data Explorer |
| `G P` | Go to Patient Timelines |

## Cohort Builder

| Shortcut | Action |
|----------|--------|
| `Ctrl/⌘ + S` | Save cohort definition |
| `Ctrl/⌘ + Enter` | Save and close |
| `Ctrl/⌘ + G` | Open generate dialog |
| `Alt + R` | Add inclusion rule |
| `Alt + C` | Add criteria to current rule |

## Vocabulary Browser

| Shortcut | Action |
|----------|--------|
| `/` | Focus search input |
| `↑ / ↓` | Navigate search results |
| `Enter` | Open selected concept |
| `A` | Add selected concept to open concept set |
| `D` | Toggle descendants flag |

## Data Explorer

| Shortcut | Action |
|----------|--------|
| `1–5` | Switch between tabs (Overview, Conditions, Drugs, Measurements, DQD) |
| `F` | Focus filter/search input |
| `E` | Export current view to CSV |

## Patient Timeline

| Shortcut | Action |
|----------|--------|
| `+` / `-` | Zoom in / zoom out on timeline |
| `←` / `→` | Pan timeline left / right |
| `0` | Reset zoom to full history |
| `F` | Toggle filter panel |

## General UI

| Shortcut | Action |
|----------|--------|
| `Ctrl/⌘ + Z` | Undo (in form editors) |
| `Tab` | Move to next form field |
| `Shift + Tab` | Move to previous form field |
| `Space` | Toggle checkboxes, activate buttons |
| `Enter` | Submit focused form |
