## Service Interaction Patterns

### Observability Flow
```
Docker containers в”Җв”Җв–ә Alloy в”Җв”Җв–ә Loki в”Җв”Җв–ә Grafana
Host metrics в”Җв”Җв–ә Node Exporter в”Җв”Җв–ә Prometheus в”Җв”Җв–ә Grafana
Container metrics в”Җв”Җв–ә cAdvisor в”Җв”Җв–ә Prometheus в”Җв”Җв–ә Grafana
```

### Data Flow (Enterprise)
```
OMOP CDM (PG17) в”Җв”Җв–ә Superset (BI dashboards)
                в”Җв”Җв–ә DataHub (catalog + lineage)
                в”Җв”Җв–ә n8n (automated ETL + quality checks)

Parthenon API в”Җв”Җв–ә n8n (workflow triggers)
              в”Җв”Җв–ә Superset (embedded analytics)
```

### Authentication Flow (Enterprise)
```
User в”Җв”Җв–ә Traefik в”Җв”Җв–ә Authentik в”Җв”Җв–ә OIDC/SAML в”Җв”Җв–ә Service
                                               в”ңв”Җ Grafana
                                               в”ңв”Җ Superset
                                               в”ңв”Җ n8n
                                               в”ңв”Җ DataHub
                                               в””в”Җ Parthenon
```
