# Phase 13.1 Plan 02: FinnGen Schema Isolation Migration Summary

**Single-transaction PG17 migration isolating FinnGen into its own schema: CREATE SCHEMA finngen, SET SCHEMA + RENAME on 6 tables, CREATE TABLE finngen.endpoint_definitions with 11-column typed projection, 5,161-row INSERT-SELECT + DELETE round-trip out of app.cohort_definitions, DROP COLUMN coverage_profile, GIN index on expression_json, finngen_endpoint_name FK split on endpoint_generations, HIGHSEC grants, and a fully functional down() that rehydrates from the Phase 13 snapshot.**
