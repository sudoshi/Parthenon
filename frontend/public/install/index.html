Install Parthenon

Unified Outcomes Research Platform on OMOP CDM

Deploy the Parthenon Community stack with a single command. The Acropolis installer checks the machine, writes local configuration, loads optional demo data, and starts the platform through a guided flow.

macOS + Linux
brew tap sudoshi/parthenon
brew install parthenon-installer
parthenon-install

Requires Homebrew. Works on macOS (Intel + Apple Silicon) and Linux.

Ubuntu / Debian
sudo snap install parthenon-installer --classic
parthenon-install

Also available as .deb on the releases page.

Windows
winget install Acumenus.ParthenonInstaller
parthenon-install

Requires WSL 2 and Docker Desktop.

or use the install script
curl -fsSL https://parthenon.acumenus.net/install.sh | sh
Requirements
8 GB RAM · 20 GB disk · Docker & Docker Compose
Linux (x64/arm64), macOS (Apple Silicon/Intel), or Windows via WSL 2
