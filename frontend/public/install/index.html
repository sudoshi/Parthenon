Install Parthenon

Unified Outcomes Research Platform on OMOP CDM

Deploy the full Parthenon stack with a single command. The Acropolis installer provisions infrastructure, configures SSO, loads OMOP vocabularies, and starts all services — community or enterprise — in one guided flow.

Download for Linux
curl -fsSL https://parthenon.acumenus.net/install.sh | sh

Having trouble? Download directly: Linux · macOS · Windows (WSL)

Requirements
8 GB RAM · 20 GB disk · Docker & Docker Compose
Linux (x64/arm64), macOS (Apple Silicon/Intel), or Windows via WSL 2
